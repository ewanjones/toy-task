# Toys

## Installation

Create and activate a Python 3.11 virtual environment using poetry:

```sh
pip install poetry
poetry env use 3.11.4
poetry install
```

## Testing and linting

Run the test suite with:

```sh
poetry run pytest
```

Run mypy with:

```sh
poetry run mypy
```

Run flake 8 with:

```sh
poetry run flake8
```
